# Grameen-Light
AI-Powered Smart Village Streetlight Monitoring & Energy Audit System

## Overview
Grameen-Light is a smart village Android application designed to improve public safety and reduce electricity wastage by enabling citizens to report faulty streetlights and helping Panchayat authorities manage maintenance efficiently.

The platform acts as a digital bridge between villagers and local governance through real-time monitoring, analytics, and AI-powered assistance.

---

## Features

### Citizen Features
- Report faulty streetlights
- Upload complaint images
- Live complaint tracking
- Nearby pole monitoring
- Voice complaint support

### Admin Features
- Complaint management dashboard
- Maintenance assignment
- Real-time analytics
- Energy-saving reports
- Heatmap visualization

### AI Features
- AI-powered issue classification
- Predictive maintenance
- Smart chatbot assistant
- Voice-to-complaint conversion
